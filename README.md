🤖 NEXUS: AI Voice-Controlled Robotics Interface

An advanced browser-based robotic control system powered by voice commands, real-time animation, and modern Web APIs — built entirely with pure HTML, CSS, and JavaScript (no frameworks, no dependencies).

🌟 Overview

NEXUS is a fully interactive AI-inspired robot interface that responds to voice commands and user inputs with realistic animations and speech output.

It simulates a futuristic robotic assistant inside your browser using:

🎤 Voice Recognition (Web Speech API)
🔊 Text-to-Speech (SpeechSynthesis API)
🎨 Real-Time Canvas Animation (60 FPS)
⚡ Zero Dependencies (Single HTML File)
✨ Features
🎙️ Voice-controlled robot (Speak commands)
🤖 Fully animated robot using Canvas API
🔊 Robot speaks responses (Text-to-Speech)
🧠 Smart keyword-based command system
🎮 Interactive control buttons (Walk, Rotate, Dance, etc.)
💬 Live speech bubble responses
📜 Command log with timestamps
🌌 Cyberpunk animated background
⌨️ Manual text input support
⚡ Runs completely in browser (No backend)
🧠 Supported Voice Commands
Command	Action
hello / hi	Robot greets
how are you	Robot responds
dance	Robot dances
walk	Robot walks
rotate	Robot spins
scan	Robot scans
stop	Robot stops
wave	Robot waves
no	Robot shakes head

💡 You can also type commands manually using the input box.
